To get running at home:

(1) If you just install clojure zip
  navigate to clojure directory (~.../clojure-1.7.0)
  cd into directory
  (Make sure to place tictactoe.clj in this directory)
  to run clj file: java -cp clojure-1.7.0.jar clojure.main tictactoe.clj
  
(2) If using leiningen package
  -Install leiningen to path
  -to start lein, type: lein repl
  -in the shell, type: (load-file "tictactoe.clj")
